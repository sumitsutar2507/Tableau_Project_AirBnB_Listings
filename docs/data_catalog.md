# 📊 Data Catalog – AirBnB Listings Project

## 📌 Overview
This dataset contains AirBnB listing information used for visualization and analysis in Tableau.

---

## 🗂 Listings Table

| Column Name | Description |
|------------|------------|
| listing_id | Unique ID of listing |
| zipcode    | Area zipcode |
| bedrooms   | Number of bedrooms |
| price      | Price per night |
| latitude   | Location latitude |
| longitude  | Location longitude |

---

## 🗂 Calendar Table

| Column Name | Description |
|------------|------------|
| listing_id | Listing ID |
| date       | Date |
| price      | Price per day |

---

## 🗂 Reviews Table

| Column Name | Description |
|------------|------------|
| listing_id | Listing ID |
| review_date| Review date |

---

## 🔗 Relationships
- `listing_id` is used to join all tables  

---

## 🎯 Purpose
- Analyze pricing trends  
- Study revenue patterns  
- Understand geographic distribution
